The Pay Model A Strategic Matrix Simulation
Build the Matrix
Executive Classroom Exercise

Reconstruct the Pay Model.

Build the strategic compensation matrix exactly as Milkovich, Newman and Gerhart present it. Four policy choices, sixteen techniques arranged in four sub-columns per row, and three objectives that span the system. There is no shortcut — the matrix has to be built one defensible cell at a time.

Learning Objectives

Objective 01 Reconstruct the four policy choices that shape any compensation system.
Objective 02 Map the sixteen techniques to the policy they operationalise.
Objective 03 Connect the matrix to the three objectives the system must serve.

How to build the matrix

The elements pool is randomised. Elements appear in no particular order — by design. The point of the exercise is to think, hypothesise, place, and revise. Use the ↻ Shuffle button at any time to randomise the pool again if the current order is steering your thinking.

Tap any tile in the pool to select it. The tile turns red and the matrix cells where it could legitimately go light up.

Tap a cell to place the tile in that position. To move a placed tile, tap it to return it to the pool, then place it elsewhere.

Tap the small "i" on any tile to open its concept card — definition, why it matters, an example, and a reflective question. Tiles are designed to be read, not just sorted.

Filter the pool using the buttons at the top — Policies, Techniques, or Objectives — to work through the matrix one category at a time, or keep them all visible and decide as you go.

Phase 02 · Reflection

Pause to think.

A short written response to close the exercise. The matrix is a map; the reflection is what makes it yours.

The Pay Model insists that policies, techniques, and objectives are connected. Pick one of the four policies and explain — in your own words — how a misalignment between its techniques and the objectives would show up in a real organisation.
